"# PhoneStore - Landing Page\n\n## Описание\n\nЭтот проект представляет собой одностраничный лендинг для продажи мобильных телефонов. Лендинг разработан с использованием HTML, CSS (Tailwind CSS) и JavaScript.\n\n## Структура проекта\n\n*   `index.html`: Основной HTML-файл с разметкой лендинга.\n*   `script.js`: Файл JavaScript, содержащий интерактивную логику (обработка кликов, валидация форм, модальные окна, плавная прокрутка).\n*   `README.md`: Этот файл с описанием проекта.\n\n## Функциональность\n\n*   **Шапка (Header):** Логотип, навигационное меню и кнопка \"Заказать звонок\".\n*   **Главный раздел (Hero):** Заголовок, подзаголовок, изображение телефона и кнопка \"Смотреть каталог\".\n*   **Каталог (Catalog):** Список телефонов с изображением, названием, описанием, ценой и кнопкой \"Купить\".\n*   **О нас (About Us):** Информация о магазине.\n*   **Контакты (Contacts):** Информация о контактах и форма обратной связи.\n*   **Подвал (Footer):** Копирайт.\n\n## Используемые технологии\n\n*   HTML\n*   CSS (Tailwind CSS)\n*   JavaScript\n\n## Запуск проекта\n\n1.  Откройте файл `index.html` в вашем браузере.\n\n## Дополнительная информация\n\n*   Проект разработан с использованием Tailwind CSS для упрощения стилизации.\n*   JavaScript используется для обработки событий (кнопки, формы) и отображения модальных окон.\n*   Лендинг адаптивен для просмотра на различных устройствах.\n"
